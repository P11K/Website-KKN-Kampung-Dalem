## üè° Deskripsi Proyek
Website ini dikembangkan sebagai **portal informasi dan administrasi kegiatan KKN** di **Kampung Dalem**.  
Proyek ini bertujuan untuk memudahkan masyarakat dan peserta KKN dalam mengakses informasi mengenai:
- Profil Kampung Dalem  
- Data penduduk & posyandu  
- Informasi kegiatan KKN  
- Dokumentasi & berita kampung  

Website ini memiliki dua sisi utama:
- **Halaman Publik:** menampilkan informasi untuk pengunjung.  
- **Panel Admin:** memungkinkan pengelola untuk mengedit, menambah, atau menghapus data kampung.
